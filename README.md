# 🎀 今天吃什么？午餐大转盘

一个可爱的午餐选择器，帮你和闺蜜解决「中午吃什么」的终极难题！

## ✨ 功能

- **三步筛选**：选肉类 → 选辣度 → 选汤/干，逐步缩小范围
- **大转盘抽签**：筛选后进入转盘，随机决定最终答案
- **纸屑特效**：转出结果时撒花庆祝
- **历史记录**：记录最近 10 次结果，避免重复
- **自定义菜单**：可以自由编辑菜品数据库
- **支持手机**：手机和电脑都能玩

## 🚀 如何使用

直接打开 [GitHub Pages 链接] 即可使用，无需安装任何东西。

## 🛠 自定义

点击页面底部的「⚙️ 自定义菜单」按钮，按格式编辑菜品：

```
emoji 菜名 | 肉类 | 辣度 | 汤类
```

例如：
```
🍛 黄焖鸡米饭 | 鸡 | 微辣 | 干
🍲 酸菜鱼 | 鱼 | 辣 | 汤
```

标签说明：
- 肉类：牛 / 鸡 / 猪 / 鱼 / 素
- 辣度：辣 / 微辣 / 不辣
- 汤类：汤 / 干
